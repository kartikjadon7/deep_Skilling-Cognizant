# Mockito

This folder contains the solutions for the **Mockito Exercises** completed as part of the **Cognizant Digital Nurture Java FSE Program**.

## Exercises Completed

### Exercise 1: Mocking and Stubbing

* Created a mock object using Mockito.
* Stubbed method behavior using `when()` and `thenReturn()`.
* Tested a service class without calling the real external API.
* Verified the returned data using JUnit assertions.

### Exercise 2: Verifying Interactions

* Created a mock object using Mockito.
* Verified method invocation using `verify()`.
* Ensured that the expected interaction occurred between the service and the mocked dependency.
