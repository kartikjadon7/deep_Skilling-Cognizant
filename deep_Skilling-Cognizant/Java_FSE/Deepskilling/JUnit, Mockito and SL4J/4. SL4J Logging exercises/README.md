# SLF4J

This folder contains the solutions for the **SLF4J Exercises** completed as part of the **Cognizant Digital Nurture Java FSE Program**.

## Exercises Completed

### Exercise 1: Logging Error Messages and Warning Levels

* Configured SLF4J with Logback.
* Implemented logging using `Logger` and `LoggerFactory`.
* Logged messages using the `ERROR` and `WARN` logging levels.
* Demonstrated basic application logging using SLF4J.
