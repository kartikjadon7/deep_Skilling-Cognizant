# PL/SQL Exercises

This folder contains the PL/SQL programming exercises completed as part of the **Cognizant Digital Nurture – Java Full Stack Engineer (Java FSE)** training program.

## Exercises Completed

### Exercise 1: Control Structures

#### Scenario 1: Loan Interest Discount
- Applied a 1% interest rate discount to customers above 60 years of age.
- Used loops, conditional statements and update operations.

#### Scenario 2: VIP Status
- Promoted customers with a balance greater than 10,000 to VIP status.
- Updated customer records using conditional logic.

#### Scenario 3: Loan Due Reminders
- Retrieved loans due within the next 30 days.
- Generated reminder messages using `DBMS_OUTPUT.PUT_LINE`.

---

### Exercise 3: Stored Procedures

#### Scenario 1: Process Monthly Interest
- Created a stored procedure to apply 1% monthly interest to all savings accounts.

#### Scenario 2: Update Employee Bonus
- Created a parameterized stored procedure to update employee salaries based on department and bonus percentage.

#### Scenario 3: Transfer Funds
- Created a stored procedure to transfer funds between accounts.
- Validated sufficient balance before processing the transaction.
- Used transaction control with `COMMIT` and `ROLLBACK`.
