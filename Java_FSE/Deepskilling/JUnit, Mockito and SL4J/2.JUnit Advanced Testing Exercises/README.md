# Advanced JUnit Testing

This folder contains the solutions for the **Advanced JUnit Testing Exercises** completed as part of the **Cognizant Digital Nurture Java FSE Program**.

## Exercises Completed

### Exercise 1: Parameterized Tests

* Created the `EvenChecker` class.
* Implemented parameterized tests using `@ParameterizedTest`.
* Used `@ValueSource` to test multiple input values.
* Verified even number validation using JUnit 5.

### Exercise 2: Test Suites

* Created multiple JUnit test classes.
* Grouped test classes using `@Suite` and `@SelectClasses`.
* Executed multiple test classes through a single test suite.

### Exercise 3: Test Execution Order

* Controlled the execution order of test methods.
* Used `@TestMethodOrder` and `@Order` annotations.
* Demonstrated ordered execution of JUnit tests.

### Exercise 4: Exception Testing

* Created the `ExceptionThrower` class.
* Verified expected exceptions using `assertThrows()`.
* Tested exception handling behavior in JUnit 5.
