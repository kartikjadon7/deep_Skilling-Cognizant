# Week 1 - Design Patterns and Principles

This folder contains the mandatory hands-on exercises completed during Week 1 of the Cognizant Digital Nurture 5.0 Java FSE training.

## Exercises

### 1. Singleton Pattern
**Description:**
Implemented the Singleton Design Pattern to ensure that only one instance of the Logger class is created and shared throughout the application.

**Files:**
- Logger.java
- SingletonTest.java
- Logger.png

---

### 2. Factory Method Pattern
**Description:**
Implemented the Factory Method Design Pattern to create different types of documents such as Word, PDF, and Excel using separate factory classes.

**Files:**
- Document.java
- WordDocument.java
- PdfDocument.java
- ExcelDocument.java
- DocumentFactory.java
- WordDocumentFactory.java
- PdfDocumentFactory.java
- ExcelDocumentFactory.java
- FactoryMethodTest.java
- FactoryMethod.png
