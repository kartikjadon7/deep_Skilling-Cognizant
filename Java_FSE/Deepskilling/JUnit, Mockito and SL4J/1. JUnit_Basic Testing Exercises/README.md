# JUnit Basic Testing Exercises

This folder contains the solutions for the **JUnit Basic Testing Exercises** as part of the **Digital Nurture 5.0 – Java FSE** program.

## Exercises

### Exercise 1 – Setting Up JUnit
- Created a Maven project.
- Configured JUnit 5 dependency in `pom.xml`.
- Verified JUnit setup with a basic test class.

### Exercise 2 – Writing Basic JUnit Tests
- Created a `Calculator` class.
- Implemented basic methods.
- Wrote unit tests using JUnit 5.

### Exercise 3 – Assertions in JUnit
- Practiced JUnit assertion methods:
  - `assertEquals()`
  - `assertNotEquals()`
  - `assertTrue()`
  - `assertFalse()`
  - `assertNull()`
  - `assertNotNull()`

### Exercise 4 – AAA Pattern, Test Fixtures, Setup and Teardown
- Implemented the Arrange–Act–Assert (AAA) testing pattern.
- Used `@BeforeEach` and `@AfterEach` annotations.
- Verified test execution using JUnit 5.

## Technologies Used

- Java 25
- Maven
- JUnit 5
- IntelliJ IDEA
